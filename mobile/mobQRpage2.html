How to Configure HMT-1 Using a QR Code

On the MR8300-1open the Configuration App.
Say “NAVIGATE HOME” then “MY PROGRAMS” then “CONFIGURATION”
The device will start beeping. Scan the QR code by pointing the camera so the QR code is within the target outline on the display.
When the QR code is scanned a tone will sound and you will be returned to the previous screen.
